1.Download node.js.
2.Unzip the project folder.
3.Open it on your IDE ( e.g viscose,clion etc)
4.Then in the terminal install all the dependencies by using the following command -> npm install
5.Then to run the prototype just do ->npm run dev
6.To access the software open your browser and type ->http://localhost:3000/
